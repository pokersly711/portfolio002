Installing

To install this template upload all files from www/php or www/html folder into HTTP server.
Some parts of template is based on AJAX requests and it can't work locally (without HTTP server).
It is highly recommended to use PHP version instead of HTML (if it is possible).
Please note that plugins Booking, Contact form and Newsletter form work only in PHP version.

Requirements

PHP version version of template needs installed PHP in version at least 5.x on server.
HTML version doesn't require additional modules (or settings) installed on server side.

Plugins

Booking

This project includes simple booking form ready for use.
To configure sending e-mail message you have to set some constants located in file plugin/booking/config.php.

Base settings
Name Description
BOOKING_FORM_NAME Name of recipient/sender e.g. My Company.
BOOKING_FORM_EMAIL E-mail address of recipent/sender e.g. mycompany@gmail.com.
BOOKING_FORM_SUBJECT Subject of e-mail message.
SMTP settings
Name Description
BOOKING_FORM_SMTP_HOST Your SMTP host e.g. smtp.gmail.com.
BOOKING_FORM_SMTP_USER Login to your e-mail account e.g. mycompany.
BOOKING_FORM_SMTP_PASSWORD Password to your e-mail account.
BOOKING_FORM_SMTP_PORT Your SMTP port e.g. 587.
BOOKING_FORM_SMTP_SECURE Secure connection protocol: SSL or TLS.
Messages
Name Description
BOOKING_FORM_MSG_INVALID_DATA_FIRST_NAME Text displayed when first name is empty.
BOOKING_FORM_MSG_INVALID_DATA_SECOND_NAME Text displayed when second name is empty.
BOOKING_FORM_MSG_INVALID_DATA_EMAIL Text displayed when e-mail address is invalid.
BOOKING_FORM_MSG_INVALID_DATA_PHONE Text displayed when phone number is empty.
BOOKING_FORM_MSG_INVALID_DATA_DATE Text displayed when booking date is invalid.
BOOKING_FORM_MSG_INVALID_DATA_MESSAGE Text displayed when message is empty.
BOOKING_FORM_MSG_SEND_OK Text displayed when e-mail was sent.
BOOKING_FORM_MSG_SEND_ERROR Text displayed when e-mail cannot be sent.

FAQ

How to enable Google Maps?

To enable Google Maps is necessary to generate special API key. Google restricts displaying maps for free users to 25000 per day, but it is possible to increase this value by purchasing Premium Plan. More info about this is availabe here: http://googlegeodevelopers.blogspot.com.au/2016/06/building-for-scale-updates-to-google.html

To run Google Maps you have to follow these steps:

  1. Go to under URL https://developers.google.com/maps/documentation/javascript/get-api-key#get-an-api-key and generate API key.
  2. [PHP] Open file include/footer.php, find line:
    <script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false"></script>
    and replace it to:
    <script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false&key=YOUR_API_KEY"></script>
    where YOUR_API_KEY is a valid API key generated in first step.
  3. [HTML] Open file in which Google Maps has to be embedded, find line:
    <script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false"></script>
    and replace it to:
    <script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false&key=YOUR_API_KEY"></script>
    where YOUR_API_KEY is a valid API key generated in first step.
How to change colors in template?

All colors used in template are defined in file style/base.css in section Colors.
Above section contains three subsections: Text color, Background color and Border color.

What dimension of images are used?

Template uses images in below dimensions (in pixels):

  • 100x100 - used in Latest posts and Most commented widgets,
  • 350x350 - used in Team list component,
  • 460x306 - used in Gallery component,
  • 460x678 - used in Gallery component,
  • 760x506 - used in Blog,
  • 960x780 - used as background of 50x50% (flex) sections,
  • 1920x500 - used as background of page header,
  • 1920x750 - used in Revolution slider.
How to run contact form in PHP version?

To enable contact form in PHP version please follow below steps:

  1. Copy files:
    php/plugin/contact-form to html/plugin/contact-form
    php/library/PHPMailer to html/library/PHPMailer
    php/library/functions.php to html/library/functions.php
    php/library/EmailTemplate.class.php to html/library/EmailTemplate.class.php
  2. Open file in which you plan to use contact form, go to scripts section and add line:
    <script type="text/javascript" src="plugin/contact-form/jquery.contactForm.js"></script>
  3. Open file html/script/public.js and add line:
    $('.template-component-contact-form').contactForm();
